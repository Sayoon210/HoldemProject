# 디자인 및 UI/UX 사양 문서 (Design Specifications)

이 문서는 게임의 시각적 요소와 사용자 인터페이스 구성을 다룹니다.

## 1. 디자인 컨셉
- **Aesthetic**: **Noir/Dramatic Casino**. 배경은 완전한 블랙(#000000)으로 처리하며, 테이블 위로 **강렬한 스포트라이트(Spotlight)**가 내리쬐어 테이블과 플레이어 영역만 강조되는 연출.
- **Color Palette**: 테이블(다크 그린 또는 딥 레드), 카드 및 토큰(고대비 컬러).

## 2. 주요 UI 구성 요소
- **Game Table**: 
    - **반원형 또는 반타원형(Semi-circle / Semi-oval)** 레이아웃.
    - **플레이어 (6명)**: 반원의 곡면부와 측면을 따라 6명이 자연스럽게 둘러앉음.
    - **커뮤니티 카드**: 테이블의 직선부(지름/변) 쪽에 5장이 가로로 배치.
- **Visual Feedback**:
    - **조커(Joker) 마커**: 현재 판의 시작 플레이어 슬롯 옆에 배치되는 아이콘/표식.
    - **토큰 스택**: 각 플레이어의 자리(UI 슬롯)에 보유한 토큰이 시각적으로 쌓여 있는 모습 연출.
    - **베팅 영역**: 플레이어가 베팅한 토큰은 플레이어 슬롯 바로 앞(테이블 안쪽)으로 이동하여 배치.
- **Player Slots**: 플레이어 이름, 보유 토큰(텍스트+그래픽), 2장의 핸드(Hole cards) 배경 주위에 배치.
- **Card Assets (Sprite Sheet)**:
    - **통합 텍스처(Texture Atlas)**: 52장의 카드를 하나의 이미지 파일(`cards_spritesheet.png`)로 관리.
    - **렌더링 방식**: CSS의 `background-position` 또는 Canvas/WebGL의 UV 좌표를 사용하여 특정 카드를 추출하여 표시.
    - **장점**: HTTP 요청 수 감소, 에셋 로딩 속도 최적화, 애니메이션 처리 용이.
- **Community Cards**: 플랍, 턴, river 카드가 배치되는 중앙 영역.
- **Betting Controls**: 체크, 레이즈, 폴드 버튼 및 레이즈 금액 조절 슬라이더.
- **Pot Information**: 현재 판돈(Pot Size) 및 이월된 토큰 표시.

## 3. 사용자 경험(UX) 흐름
- 게임 입장 -> 테이블 매칭 -> 게임 진행(베팅 라운드) -> 승자 발표 및 연출.
